Natural Language Processing

Methods in Language Technology

Course description

This course gives an in-depth study in basic methods and practical tools for basal language technology. It covers both rule-based techniques, such as phrase structure grammar, and approximations with a starting point in machine learning, such as vector space semantics and classification.

The course will take a look at some applications of methods for issues within language technology such as tagging, parsing and text classification (such as sentiment analysis).

This is a course with a strong practical component, with use of relevant tools and projects with written rapports.

Theory topics

Machine Learning approaches Document vectors Classification & vector spaces Clustering Lexical semantics & word vectors Logistic regression Sequences Dependency syntax Parsing Machine translation Interactive systems
